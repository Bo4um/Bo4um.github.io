Наследование и полиморфизм

Для предыдущего задания по системе контроля версий Git выполнить следующую модификацию приложения:

  1. Добавить отдельный класс Component, в котором описать строковое поле name (наименование модели комплектующего), а также get-тер и set-тер для этого поля. Сгенерировать метод toString().
  2. Описать в классе Component поле cost (стоимость в копейках), а также get-тер и set-тер для этого поля.
  3. Описать в классе Component метод totalCost(), возвращающий результат выполнения метода getCost().
  4. Сделать все ранее созданные классы, описывающие как компьютерные комплектующие так и сам компьютер, подклассами класса Component. Из каждого такого класса убрать поле cost и get-тер и set-тер для этого поля, так как это поле будет унаследованным от базового класса.
  5. Добавить к методам totalCost() всех наследников класса Component, в которых этот метод явно переопределяется, аннотацию @Override.
  6. Модифицировать метод main() так, чтобы он использовал наименование модели комплектующих.
